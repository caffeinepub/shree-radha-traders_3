# Specification

## Summary
**Goal:** Build a luxurious Indian wedding gift packaging website for Shree Radha Traders with admin dashboard for content management and customer inquiry handling.

**Planned changes:**
- Create multi-admin authentication system with secure login
- Build admin dashboard with product management (upload/delete photos, edit descriptions, manage categories, update pricing)
- Implement admin controls for homepage banners and inquiry management
- Design hero section with wedding-themed background, heading "Premium Wedding Gift Packaging for Grand Celebrations", subheading "Bulk Orders | Pan India Delivery", and CTAs for "View Collection" and "Order on WhatsApp"
- Create About Us section highlighting Shree Radha Traders from Churu, Rajasthan
- Build product gallery with categories (Fancy Potlis, Designer Envelopes, Wedding Pouches, Jute Bags, Return Gifts, Corporate Gifting) in grid layout with zoom functionality
- Add "Why Choose Us" section with four benefits: Premium Quality Material, Wholesale Prices, Fast Delivery, Trusted by Wedding Planners
- Implement contact section with click-to-call phone, WhatsApp direct chat, Instagram link, inquiry form (Name, Phone, Message), and embedded Google Map for Churu, Rajasthan
- Design footer with social media icons, quick links, and copyright text
- Add floating WhatsApp button visible across all pages
- Apply royal color palette (Maroon primary, Gold accent, Cream and Soft Pink complementary) throughout
- Ensure full mobile, tablet, and desktop responsiveness

**User-visible outcome:** Visitors can browse a luxurious wedding gift packaging catalog with multiple product categories, contact Shree Radha Traders via WhatsApp/phone/form, and view company information. Admins can log in to manage products, banners, and customer inquiries through a comprehensive dashboard.
